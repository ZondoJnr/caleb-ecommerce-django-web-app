API Documentation

Use the endpoints below to interact with Caleb's eCommerce API via Postman or any HTTP client.

Authentication

  • Login: /login/
  • Register: /register/

Stores

  • List Stores: /api/stores/
  • Create Store: /api/stores/create/ (POST)
  • Products by Store Name: /api/stores/<store_name>/products/

Products

  • Create Product: /api/products/create/ (POST)

Reviews

  • List All Reviews: /api/reviews/

Use tools like Postman or cURL to make authenticated API calls. Ensure to include CSRF tokens or authentication headers where required.
